# alertlens-templates
Templates for creating AlertLens jobs

Currently available:
- AWS CloudWatch
  - EC2 Standard and Nitro Instances
  - EBS
  - RDS
  - SQS
- Prometheus
  - Node Exporter
  
Refer to https://boltanalytics.com/docs/docs/new-job/template/ on how to deploy a template in Alert Lens.
